CariAnak.id

Description

Aplikasi CariAnak.id ini merupakan Aplikasi yang akan di gunakan untuk memudahkan Anak Magang dalam mencari tempat magang di beberapa Perusahan, dan pihak Perusahaan juga dapat merekrut / memberikan undangan untuk kegiatan magang kepada Kandidat Magang / Siswa. Aplikasi ini saya buat pada tahun 2020 pada saat saya sedang melakukan kegiatan Praktik Kerja Industri di PT. For It Asta Solusindo (Sekolahan.id) bersama 1 teman saya. Dalam pembuatan aplikasi ini saya memakai Bahasa PHP dan menggunakan Framework Laravel, dan dalam project ini saya merombak ulang / membuat ulang dari 0 bersama tim saya (tim saya berjumlah 2 orang termasuk saya). Dan saya juga berperan sebagai FullStack Web Developer (membagi tugas dengan teman saya), dan merancang database nya di MySQL
