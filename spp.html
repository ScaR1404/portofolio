SPP.id

Description

Aplikasi SPP.id ini merupakan Aplikasi pembayaran SPP Sekolah yang di lakukan oleh petugas / admin di bagian Tata Usaha (belum melakukan online payment) karna aplikasi ini hanya digunakan untuk pendataan dan rekap data SPP. Aplikasi ini saya buat pada tahun 2020 pada saat saya masih duduk di bangku SMK, dan Aplikasi ini merupakan tugas Individu untuk memenuhi syarat kelulusan. Dalam pembuatan aplikasi ini saya memakai Bahasa PHP dan menggunakan Framework CodeIgniter. Dan saya juga berperan sebagai FullStack Web Developer, merancang alur diagram nya, serta merancang database nya di MySQL
